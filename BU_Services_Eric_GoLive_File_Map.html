Power Automate — Sunday 7PM Scheduled Flow

~2 hrs Éric Requires Step 1 complete — SOQL queries need BU_Initiative__c to exist
Build the scheduled PA flow that runs every Sunday at 7:00 PM ET. It queries Salesforce (Closed Won by initiative, region, entity), reads the SharePoint utilisation tracker, and writes results to actuals.json. Critical: the flow must merge new data into the existing JSON — it must never overwrite Jeremy Irwin’s manual Nortech entries or Jon’s gate/risk status.
BU_Services_Power_Automate_Build_Guide.docx
Complete PA build guide. Contains: system overview, prerequisites checklist, all 5 SOQL query templates with exact syntax, step-by-step PA flow structure, actuals.json field mapping table, SharePoint file locations, testing checklist, troubleshooting table, and annual reset instructions. This is your primary build spec for Step 2.

Annotated reference for all 110 fields in actuals.json. Colour-coded by owner: which fields PA writes automatically, which Jeremy Irwin enters manually, which Jon Eustace updates. Essential companion when building the PA flow to know exactly which fields to write and which to preserve.
