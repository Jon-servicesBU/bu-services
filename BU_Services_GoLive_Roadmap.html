PA
Sunday 7PM ET Scheduled Flow — the core automation
Éric · ~2 hrs

This is the central automation that powers the entire system. It runs every Sunday at 7PM ET, queries Salesforce for all revenue and pipeline data, assembles a JSON payload, and writes actuals.json to SharePoint. Every downstream output — Command Centre, deck, email — depends on this flow.

Closed Won queries (6)
One Get Records action per initiative (Mining, Pipeline, Powerline, Nuclear, 3X) + one for Base Business. Filter: StageName = 'Closed Won' AND CloseDate = THIS_YEAR AND BU_Initiative__c = '[value]'. Sum Amount. Writes rev_mining, rev_pipeline etc. to actuals.json.
Pipeline queries (3)
Bookings: all Opps created this year. Commit: probability ≥ 75%. Upside: probability 25–74%. Writes bookings_nuc, commit_nuc, upside_nuc to actuals.json. Nortech equivalents remain manual — Nortech not in Salesforce.
Regional queries (10)
Closed Won + pipeline per region (HQ, Montréal, Central, Edmonton, Vancouver) using the Region field API name confirmed in Step 1. Writes region_hq, region_montreal, etc. Final Compose action assembles full JSON payload and writes actuals.json to SharePoint.
